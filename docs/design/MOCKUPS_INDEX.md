# Uni-Connect — Design Mockups Index

## HTML Prototype Files

This folder contains all HTML prototype mockups for the Uni-Connect platform. These are interactive design mockups that show the UI/UX structure and layout.

**Source Location:** `../../project to be know/` (HTML files)

### Available Mockups

#### Authentication & Account Management
- `login.html` — User login page with email/password form
- `register.html` — New user registration form
- `admin.html` — Admin dashboard and user management interface

#### Core Platform Pages
- `index.html` — Landing page / home page
- `dashboard.html` — Main user dashboard
- `profile.html` — User profile page (personal details, stats)
- `user-profile.html` — Public user profile view (visiting another user's profile)

#### Social Features
- `create-post.html` — Form for creating new posts/discussions
- `single-post.html` — Individual post view with comments/replies
- `sessions.html` — Study sessions management page
- `notifications.html` — User notifications center/inbox

#### Gamification & Engagement
- `points.html` — Points and rewards dashboard
- `leaderboard.html` — User rankings and leaderboard
- `settings.html` — User settings and preferences

## How to Use These Files

1. **For UI Reference:** Open any `.html` file in a browser to see the mockup
2. **For Design Review:** Reference the wireframe `*.png` files for layout specifications
3. **For Implementation:** Use these as reference for CSS styling and HTML structure
4. **For Documentation:** Screenshot mockup pages and add to project reports

## Notes

- These are static HTML mockups (not functional)
